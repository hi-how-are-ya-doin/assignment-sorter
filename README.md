# assignment-sorter
A Python desktop app that automatically sorts assignment files into folders by class and type.


**REQUIREMENTS**

1. **Python 3**

Windows/Mac: download from python.org

Chromebook: sudo apt install python3

2. **tkinter** (the UI library)

Usually comes with Python automatically

Chromebook: sudo apt install python3-tk

3. **watchdog** (for the auto-watch feature)

pip3 install watchdog --break-system-packages
